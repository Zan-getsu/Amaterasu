Google Cloud flight checklist

Prepare your OAuth client before you press Continue with Google.

Use the same Google account for the Cloud project, test user, login hint, and final authorization. On mobile, switch Google Cloud Console to desktop mode first.

Open Console
  1. Secure the accountEnable 2-Step Verification, then create or select a Google Cloud project.
  2. Enable Drive APIGo to APIs & Services, open Library, search Google Drive API, and enable it.
  3. Set consent screenOpen OAuth consent screen, click Get Started, enter an app name, choose External, add your Gmail, and save.
  4. Add test userIn OAuth consent screen, open Audience, add the same Gmail under Test users.
  5. Create Web clientGo to Credentials, create an OAuth client ID, and choose Web application.
  6. Paste this redirect URICopy the URI above into Authorized redirect URIs, then create the client.
  7. Download credentialsDownload the OAuth client JSON and rename it exactly to credentials.json.
  8. Authorize hereUpload credentials.json below, enter the same Gmail as the login hint, continue, review Google's warning, and allow access.
Only continue past Google's unverified-app warning when the Cloud project and OAuth credentials belong to you. Keep credentials.json, token.pickle, and this private bot link away from public repos and shared chats.
